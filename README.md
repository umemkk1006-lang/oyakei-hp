# 親子経済ラボ - 公式ホームページ

![親子経済ラボ](images/logo.png)

## プロジェクト概要

**親子経済ラボ**は、親子で経済・判断・生き方を考えるための静かで信頼できる思考の場所です。

このウェブサイトは「売るためのサイト」ではなく、研究・思想を中心に据え、考える力・整理する力を大切にする思想の拠点となることを目指しています。

### サイトの基本思想

- 答えを提示するのではなく、考える力を育てる
- 正解のない時代に、「今ここ」で考え直すための整理を大切にする
- 研究と思想が活動の中心
- 有料化については前面に出さず、信頼と思想の蓄積の先に自然に存在するものとして示唆

## 推奨される次のステップ

1. **追加画像の配置**
   - ユーザーから提供された画像素材を`images/`フォルダに追加

2. **追加深掘相談・話して相談のリンク作成**
   - Googleフォーム、Calendly、または専用の予約システムを準備

3. **コンテンツの拡充**

   - 活動実績やケーススタディの追加

4. **SEO対策**
   - メタディスクリプション・OGPタグの最適化
   - 構造化データの追加
   - サイトマップの作成

5. **アクセス解析の導入**
   - Google Analyticsまたはプライバシー重視の解析ツールの導入

## デザインコンセプト

### カラーパレット

- **ミントグリーン** (`#B8E6D5`)：メインカラー、安心感
- **ピーチ** (`#FFD4B8`)：アクセントカラー、温かさ
- **レモンイエロー** (`#FFF4C2`)：サブアクセント、明るさ
- **クリーム** (`#FFF9F0`)：背景色、やさしさ
- **ホワイト** (`#FFFFFF`)：コンテンツ背景
- **チャコールグレー** (`#4A4A4A`)：メインテキスト

### タイポグラフィ

- **フォントファミリー**: Noto Sans JP（Google Fonts）
- **フォントウェイト**: 
  - Light (300): 本文
  - Regular (400): ナビゲーション、リンク
  - Medium (500): 見出し
  - SemiBold (600): 強調

### スペーシング

- **余白を多く取る**：視覚的な圧迫感を避け、読みやすさを優先
- **セクション間のスペース**：3-5rem
- **カード内のパディング**：2-3rem
